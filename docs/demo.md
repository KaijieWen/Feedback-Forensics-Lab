# Demo script (3–5 minutes)

1. **Dashboard**: Highlight top clusters + recent cases.
2. **Search**: Query “cache purge slow” and open a result.
3. **Case detail**: Show priority score, clarifying question, and juror votes in JSON.
4. **Ingest live**: Submit a new feedback item; watch status move from `queued` → `processing` → `ready`.
5. **Cluster view**: Open the cluster and point out similar member cases.
